# ADSORFIT Frontend

This is the React + TypeScript frontend for ADSORFIT.

## Development

Install dependencies:
```bash
npm install
```

Run development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

## Configuration

The frontend connects to the FastAPI backend at the URL specified in `.env`:
- `VITE_API_BASE_URL`: Backend API base URL (default: http://127.0.0.1:8000)

## Technology Stack

- **React 18** - UI framework
- **TypeScript** - Type-safe development
- **Vite** - Fast build tool and dev server
- **CSS3** - Modern styling with custom design system
